# 🤖 Проект: Personal Finance Bot (Telegram + GPT)

## 📁 Архитектура проекта (Модули)

[cite_start]Для обеспечения высокого качества кода[cite: 18], проект разделен на логические части:

1.  **`main.py`**: Главный файл запуска. Содержит инициализацию бота и все обработчики сообщений (handlers).
2.  **`database.py`**: Модуль для работы с базой данных (SQLite). [cite_start]Отвечает за хранение доходов, расходов и настроек пользователя[cite: 16, 18].
3.  **`gpt_service.py`**: Модуль для связи с OpenAI API. Обрабатывает запросы на получение финансовых советов.
4.  [cite_start]**`keyboards.py`**: Описание всех кнопок (Reply и Inline) для обеспечения качественного UX[cite: 18].
5.  [cite_start]**`utils.py`**: Вспомогательные функции (генерация графиков, форматирование текста, работа с изображениями)[cite: 18].

## 🛠 Описание функций и модулей

### 1. Обработка команд и сообщений (`main.py`)
* [cite_start]**`/start`**: Отправляет приветственное сообщение и инструкции[cite: 18]. Устанавливает «уникальную тему» через текстовое оформление.
* [cite_start]**Обработка текста**: Бот анализирует ввод пользователя (например, "Обед 500") с помощью регулярных выражений или GPT, чтобы автоматически распределить траты по категориям[cite: 32].
* [cite_start]**Обработка фото**: При получении изображения (например, чека), бот подтверждает получение и имитирует/выполняет «сканирование»[cite: 18].

### 2. Умные советы через GPT API (`gpt_service.py`)
* **Функция `get_financial_advice(user_data)`**: Отправляет накопленную статистику пользователя в GPT.
* **Результат**: Пользователь получает персонализированный совет (например, «Ты тратишь слишком много на кофе, попробуй сократить этот расход на 10%»). [cite_start]Это покрывает требование по интеграции стороннего API[cite: 18].

### 3. База данных (`database.py`)
* **Таблица `users`**: ID пользователя, установленный бюджет.
* **Таблица `transactions`**: ID, тип (доход/расход), сумма, категория, дата.
* [cite_start]Это позволяет боту «помнить» историю и выдавать точные ответы на основе сценария[cite: 18].

## 🌊 UX Flow (Путь пользователя)

1.  **Регистрация**: 
    * Пользователь пишет `/start`.
    * Бот выдает красивое приветствие: «Привет! Я твой AI-финансист 📈. Давай начнем с твоего бюджета».
2.  **Настройка**:
    * Кнопка **«Установить бюджет»**. Пользователь вводит сумму (например, 1000 USD).
3.  **Ежедневное использование**:
    * Пользователь нажимает кнопку **«Добавить расход»** или просто пишет «Такси 200».
    * Бот сохраняет это в БД и присылает подтверждение: «Ок! Остаток: 800 USD».
4.  **Аналитика и AI**:
    * Кнопка **«Совет от GPT»**. Бот анализирует траты за неделю и присылает текстовый отчет с рекомендациями.
    * Кнопка **«Отчет в графиках»**. [cite_start]Бот генерирует изображение (chart) и отправляет пользователю[cite: 18].
